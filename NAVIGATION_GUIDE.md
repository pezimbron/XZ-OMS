## Navigation Tips

- **Dashboard is your home base** - All key features are one click away
- **Calendar has action buttons** - No need to navigate away to create jobs
- **Use Quick Create for speed** - Much faster than manual entry for email-based jobs
- **Bookmark frequently used pages** - Direct URLs work great in browser bookmarks

## Color Coding Reference

Quick Access cards use colors to help you quickly identify features:
- **Blue** = Viewing/Monitoring (Calendar)
- **Green** = Quick Actions (AI-powered Quick Create)
- **Purple** = Management (All Jobs list)
- **Indigo** = Manual Creation (Full form)

## Mobile Access

All navigation features work on mobile devices:
- Dashboard cards stack vertically on small screens
- Calendar view is responsive
- Quick Create form works on mobile

## Keyboard Shortcuts

While there are no built-in keyboard shortcuts yet, you can use browser features:
- `Ctrl/Cmd + Click` on cards to open in new tab
- Browser back button to return to previous page
- Bookmark frequently used pages for quick access
